Thuật Toán Đệ Quy (Recursion)

Đệ quy là gì?

Đệ quy là một phương pháp giải quyết vấn đề trong đó một hàm tự gọi lại chính nó để giải quyết các bài toán nhỏ hơn cùng một dạng.

Ví dụ về Đệ quy

1. Tính giai thừa

Code mẫu:

function factorial(n) {
    // Điều kiện cơ sở
    if (n <= 1) {
        return 1;
    }
    
    // Công thức đệ quy
    return n * factorial(n-1);
}

2. Dãy Fibonacci

Code mẫu:

function fibonacci(n) {
    // Điều kiện cơ sở
    if (n <= 1) {
        return n;
    }
    
    // Công thức đệ quy
    return fibonacci(n-1) + fibonacci(n-2);
}

Các thành phần cơ bản của đệ quy

1. Điều kiện cơ sở (Base case)

Là điều kiện dừng của đệ quy, xác định khi nào thuật toán ngừng gọi chính nó.

  • Là trường hợp đơn giản nhất của bài toán
  • Có thể giải quyết trực tiếp mà không cần đệ quy
  • Luôn phải có ít nhất một điều kiện cơ sở để tránh đệ quy vô hạn

Ví dụ:

  • Giai thừa: n = 0 hoặc n = 1 → return 1
  • Fibonacci: n = 0 → return 0; n = 1 → return 1

2. Công thức đệ quy (Recursive case)

Là công thức giải quyết bài toán bằng cách chia nhỏ thành các bài toán con cùng dạng.

  • Gọi lại chính hàm đó với tham số khác
  • Phải tiến dần đến điều kiện cơ sở
  • Kết hợp kết quả của các bài toán con để có kết quả cuối cùng

Ví dụ:

  • Giai thừa: n! = n × (n-1)!
  • Fibonacci: F(n) = F(n-1) + F(n-2)
